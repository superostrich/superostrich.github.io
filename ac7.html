Overview

These texture mods replace one or more of the default aircraft skins in Ace Combat 7.

Installation

Simply place the downloaded .pak file into ..\steamapps\common\ACE COMBAT 7\Game\Content\Paks\~mods

F-104C -Metal-

This pack contains bare-metal liveries inspired by the Starfighters of the 1960's. Includes high-viz markings for Mage, Spare, and Strider squadrons; as well as a version with no squadron markings.

Download

Replaces Colors 01,04,05,& 06
