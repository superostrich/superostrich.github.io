Ace Combat 7 Modding Guides

Simple Texture Replacement Guide

This guide covers extracting textures from AC7, loading and saving them in an image editor, and packing your mod for use in-game. It's uses lightweight portable apps and python scrips.

UE4 Texture Modding Guide - by Dantofu

This guide covers texture replacement using Unreal Engine 4 asset cooking. The process is more involved, but allows for preserving the lowest mipmap level for replaced textures, and allows you to export your textures using high quality BC7 compression.

Blender Quick Start

This guide covers busing Blender to preview & edit your textures on 3D models, as well as ripping models using Ninjaripper.

Ostrich's Texture Mods

These texture mods replace one or more of the default aircraft skins in Ace Combat 7.

To install, simply place the downloaded .pak file into ..\steamapps\common\ACE COMBAT 7\Game\Content\Paks\~mods

F-2A -502-

This pack features liveries based on JASDF F-2A No. 63-8502 from the Air Development and Test Wing, as well as Flyuger from Ace Combat 6. Replaces Color 02 or 03.

There are three variants included in this download;

  • f2a-502-noemblems-cxx.pak - 502 Livery with no emblems
  • f2a-502-adtw-cxx.pak - F-2A No. 502 with JASDF Air Development Test and Wing emblems
  • f2a-flyuger-cxx.pak - Flyuger from Ace Combat 6, with Estovakian Air Force markings
