Overview

This HD texture pack for Ace Combat X aims to make this classic PSP game feel at home on modern systems. It is a complete overhaul with bespoke HD textures (not AI upscales!) for all 160 playable aircraft skins; includes all-new high resolution HUD, fonts, menu screens, briefings, and more. Additionally, installing this pack lets you easily customize your experience with skins, effects, and other fan-made mods. Made for use with the incredible PPSSPP emulator.

Installation

Automatic Installation (PPSSPP 1.9+)

  1. From the PPSSPP main menu;
     Click Load → Browse to the downloaded Zip file and open
         -or-
     Drag and Drop the downloaded Zip file onto the PPSSPP window
  2. In PPSSPP's settings menu, check 'Replace Textures' under Settings→Tools→Developer_Tools
  3. Turn off 'Texture Scaling' under Setting→Graphics→Texture_Scaling
  4. This pack includes a high resolution replacement for the English PSP system font. You can find the PSP system font, and instructions on installing it here
  5. For best results set render resolution to 4x PSP (or higher). If you have a powerful graphics card, enable postprocessing shader Supersampling AA (Gauss)

Manual Installation

Extract the contents of the zip file to:

  • US ..\Documents\PPSSPP\PSP\TEXTURES\ULUS10176
  • JP ..\Documents\PPSSPP\PSP\TEXTURES\ULJS00086
  • EU ..\Documents\PPSSPP\PSP\TEXTURES\UCES00423
Note: If you are using the portable version of PPSSPP, the install path is:
  • ..\ppsspp\memstick\PSP\TEXTURES\ULUS10176

Changing Button Prompts

By default, this pack uses XBox Controller button prompts.

To use Playstation prompts, copy the files from ..\interface\fonts\playstation into..\interface\fonts\ and overwrite when prompted.
