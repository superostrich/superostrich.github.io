Folder Location

The \effects and \aircraft folder location varies depending on whether you're using the installed or portable version of PPSSPP

Installed
C:\Users\*yourname*\Documents\PPSSPP\PSP\TEXTURES\*DiscID*

Portable
..\ppsspp\memstick\PSP\TEXTURES\*DiscID*

Refer to the table below for the DiscID

Game Region DiscID
Ace Combat X US ULUS10176
Ace Combat X JP ULJS00086
Ace Combat X EU UCES00423
Ace Combat Joint Assault US ULUS10511
Ace Combat Joint Assault JP NPJH50263
Ace Combat Joint Assault EU ULES01408
