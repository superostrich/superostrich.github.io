Overview

This pack contains remastered, 1024px textures, for all 160 playable aircraft liveries in Ace Combat X; for use with PPSSPP. More features, improvements, and UX enhancements coming soon!

Click to expand image

Installation

Simply extract the contents of the zip file to \ppsspp\memstick\PSP\TEXTURES\ULUS10176

In PPSSPP, make sure you've enabled 'Replace Textures' under Settings > Tools > Developer Tools

Note: This pack uses the same file naming convention as TugaAvenger's HD Texture pack, and all of the aircraft skins are interchangeable.

Changing Button Prompts

By default, this pack uses XBox Controller button prompts.

To use high quality Playstation prompts, replace the contents of \ppsspp\memstick\PSP\TEXTURES\ULUS10176\buttons with the contents of this file.

Alternativelty, you can delete the entire buttons folder to use the original PSP controller prompts.
