Overview

This pack contains high definition HUD textures for the North American versions of Ace Combat 04, Ace Combat 5, and Ace Combat Zero. Requires PCSX2.

Made in collaboration with Roach Typografics, Zaku, PhantomPhanatic, and Stahl.

Features

The scope of this pack is limited to replacing the in-mission HUD on Ace Combat 04, 5 and Zero. At the moment, PCSX2's texture replacement function is missing critical features (wildcarding, texture manifest, hash reduction, etc.) that would allow for a more comprehensive overhaul. If these features are implemented, we may widen the scope to cover menus, briefings, title screens, fonts, etc.

Installation

  1. Extract the contents of the zip file to: ..\PCSX2\TEXTURES\
  2. In the Texture Replacement tab of PCSX2's Graphics Settings; Check Load Textures
